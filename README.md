### Este repositório reúne os códigos, exercícios e experimentos desenvolvidos durante a disciplina Laboratório de Programação, com foco na linguagem C. O objetivo é aplicar conceitos fundamentais de algoritmos, lógica e manipulação de dados, utilizando C como ferramenta principal.
### O objetivo é registrar minha evolução na linguagem C e consolidar os principais conceitos vistos na disciplina. O repositório também serve como referência de estudo para atividades futuras, desenvolvimento acadêmico e preparação para projetos maiores.
